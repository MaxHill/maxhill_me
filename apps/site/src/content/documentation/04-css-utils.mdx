# CSS Utilities

Utility classes for spacing, borders, and layout. All utilities use logical properties for better internationalization support.

## Directional Suffixes

Utilities support directional variants using logical property naming:

- `-i` — inline (horizontal: left & right)
- `-is` — inline-start (left in LTR, right in RTL)
- `-ie` — inline-end (right in LTR, left in RTL)
- `-b` — block (vertical: top & bottom)
- `-bs` — block-start (top)
- `-be` — block-end (bottom)

## Size Tokens

Most utilities support these size scales:

- Fixed sizes: `000`, `00`, `1-15`
- Fluid sizes: `fluid-1` through `fluid-10` (responsive, scale with viewport)
- Auto: `auto` (margin only)

--- 
# Borders

Border utilities add borders to elements using the `[data-border]` attribute. Combine with `[data-variant]` for different looks.

### Default

<div data-border>[data-border]</div>

### Inline

<div data-border="i">[data-border="i"]</div>
<div data-border="is">[data-border="is"]</div>
<div data-border="ie">[data-border="ie"]</div>

### Block

<div data-border="b">[data-border="b"]</div>
<div data-border="bs">[data-border="bs"]</div>
<div data-border="be">[data-border="be"]</div>

### Border Variants

Use `[data-variant]` to apply different border styles:

<div data-border data-variant="strong">
  [data-border] [data-variant="strong"]
</div>

<div data-border data-variant="accent">
  [data-border] [data-variant="accent"]
</div>

Available variants: `strong`, `accent`, `danger`, `success`, `neutral`, `info`, `warning`

# Gaps

Gap utilities control spacing between flex or grid items. Works with any layout container that supports the `gap` property.

<div
  style="display:grid;"
  class="box"
  data-gap="3"
  data-variant="surface"
  data-width="breakout"
>
  <div class="box">data-gap="3"</div>
  <div class="box">data-gap="3"</div>
  <div class="box">data-gap="3"</div>
</div>

<div
  style="display:grid;"
  class="box"
  data-gap="fluid-3"
  data-variant="surface"
  data-width="breakout"
>
  <div class="box">data-gap="fluid-3"</div>
  <div class="box">data-gap="fluid-3"</div>
  <div class="box">data-gap="fluid-3"</div>
</div>

Available sizes: `000`, `00`, `1-15`, `fluid-1` through `fluid-10`

# Margin

Margin utilities add space outside elements using the `[data-margin]` attribute. Supports all directional variants and includes `auto` for centering.

### All Sides

<div
  class="box"
  data-variant="surface"
  data-padded="false"
  data-width="breakout"
>
  <div data-border data-margin="1">
    [data-margin="1"]
  </div>
  <div data-border data-margin="fluid-5">
    [data-margin="fluid-5"]
  </div>
</div>

### Inline

<div
  class="box exposed-grid"
  data-variant="surface"
  data-padded="false"
  data-width="breakout"
>
  <div data-margin="i-2">[data-margin="i-2"] (left & right)</div>
  <div data-margin="is-3">[data-margin="is-3"] (left)</div>
  <div data-margin="ie-3">[data-margin="ie-3"] (right)</div>
  <div data-margin="i-fluid-3">[data-margin="i-fluid-3"]</div>
</div>

### Block

<div
  class="box exposed-grid"
  data-variant="surface"
  data-padded="false"
  data-width="breakout"
>
  <div data-margin="b-2">[data-margin="b-2"] (top & bottom)</div>
  <div data-margin="bs-3">[data-margin="bs-3"] (top)</div>
  <div data-margin="be-3">[data-margin="be-3"] (bottom)</div>
  <div data-margin="b-fluid-3">[data-margin="b-fluid-3"]</div>
</div>

Available sizes: `000`, `00`, `1-15`, `fluid-1` through `fluid-10`, `auto`

# Align

Alignment utilities for controlling justify-content and align-items using data attributes.

## Justify Content

Controls main-axis alignment (horizontal in row, vertical in column).

- `[data-justify="start"]` — align items to start
- `[data-justify="center"]` — center items
- `[data-justify="end"]` — align items to end
- `[data-justify="between"]` — space between items
- `[data-justify="around"]` — space around items
- `[data-justify="evenly"]` — even space distribution

## Align Items

Controls cross-axis alignment (vertical in row, horizontal in column).

- `[data-align="start"]` — align items to start
- `[data-align="center"]` — center items
- `[data-align="end"]` — align items to end
- `[data-align="baseline"]` — align items to text baseline
- `[data-align="stretch"]` — stretch items to fill container

# Typesetting

The `.typesetting` class creates consistent vertical rhythm for text-heavy content using CSS Grid.

## Use Cases

Apply `.typesetting` to containers with:

- Articles and blog posts
- Documentation pages
- Long-form written content
- Any content requiring consistent vertical rhythm

<card-element>
    <strong slot="title">Without .typesetting</strong>
    <div>
      # Article Title
      ## Introduction
      This is a paragraph without the typesetting class. Notice how the spacing feels inconsistent and elements appear cramped together.
      ## Main Content
      Another paragraph here to show the spacing issues when not using the typesetting utility.
    </div>
</card-element>

<card-element>
    <strong slot="title">With .typesetting</strong>
    <div class="typesetting">
      # Article Title
      ## Introduction
      This is a paragraph with the typesetting class applied. The spacing is consistent and creates a natural reading rhythm.
      ## Main Content
      Another paragraph demonstrating the improved vertical rhythm and breathing room between elements.
    </div>
</card-element>
