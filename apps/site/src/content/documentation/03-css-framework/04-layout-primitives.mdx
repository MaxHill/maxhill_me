A set of layout primitives that can be used to build interfaces with.

---
## Grid
Flexible grid layout with configurable columns and spanning capabilities.

- `.grid` — creates a grid layout (defaults to 1 column)
- `[data-cols="1-12"]` — set number of columns
- `[data-span="1-12"]` — span multiple columns
- `[data-row-span="1-12"]` — span multiple rows

Combine with `[data-gap]` for spacing between grid items,
or combine with `[data-justify]` and `[data-align]` for cell alignment.

### Basic 3 col grid
<div class="grid" data-cols="3" data-gap="2">
    <div class="box"></div>
    <div class="box"></div>
    <div class="box"></div>
</div>

### Columnd spanning
<div class="grid" data-cols="3" data-gap="2">
    <div class="box" data-span="2"></div>
    <div class="box"></div>
</div>

### Row and columnd spanning
<div class="grid" data-cols="3" data-gap="2">
    <div class="box" data-row-span="2"></div>
    <div class="box"></div>
    <div class="box"></div>
    <div class="box" data-span="2"></div>
</div>

## Collection

Responsive grid layout that automatically adapts to available space. Items wrap to new rows when needed.

- `.collection` — responsive grid with default card width
- `.collection[data-size="1-10"]` — adjust minimum card width using fluid size scale

The collection uses `auto-fill` to create as many columns as will fit. Combine with `[data-gap]` for spacing.

<section class="box collection" data-width="breakout" data-variant="surface">
  <div class="box" data-variant="surface-raised">
    .collection
  </div>
  <div class="box" data-variant="surface-raised">
    .collection
  </div>
  <div class="box" data-variant="surface-raised">
    .collection
  </div>
</section>

## Content Grid

Full-width grid layout with centered content area and optional breakout zones. Perfect for article layouts and documentation pages.

- `.content-grid` — creates three-column grid (full-width, breakout, content)
- Children get content width by default
- `[data-width="breakout"]` — wider than content, but not full-width
- `[data-width="full"]` — spans entire viewport width

<div class="box content-grid" data-gap="2" data-width="full" style="width:70%; --_content-max-width: 70%; --_breakout-max-width: 85%; " data-padded="false">
  <div class="box" data-variant="surface">
    .content-grid (content width, default)
  </div>
  <div class="box" data-width="breakout" data-variant="accent">
    .content-grid [data-width="breakout"]
  </div>
  <div class="box" data-width="full" data-variant="surface" >
    .content-grid [data-width="full"]
  </div>
</div>

## Stack

Stack components create flex layouts for arranging items vertically or horizontally.

- `.stack` — vertical stack (column, default)
- `.stack[data-direction="row"]` — horizontal stack

Combine with `[data-gap]`, `[data-align]`, and `[data-justify]` attributes for spacing and alignment.
