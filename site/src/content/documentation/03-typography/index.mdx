## Typesetting container

The `.typesetting` class is used for long-form written content like articles, blog posts, and documentation. It automatically manages vertical rhythm and spacing between text elements.

**Without .typesetting**
<div class="box" data-variant="surface">
  <h1>Article Title</h1>
  <h2>Introduction</h2>
  <p>
    This is a paragraph without the typesetting class. Notice how the
    spacing feels inconsistent and elements appear cramped together.
  </p>
  <h2>Main Content</h2>
  <p>
    Another paragraph here to show the spacing issues when not using the
    typesetting utility.
  </p>
</div>

**With .typesetting**

<div class="box typesetting" data-variant="surface">
  <h1>Article Title</h1>
  <h2>Introduction</h2>
  <p>
    This is a paragraph with the typesetting class applied. The spacing is
    consistent and creates a natural reading rhythm.
  </p>
  <h2>Main Content</h2>
  <p>
    Another paragraph demonstrating the improved vertical rhythm and
    breathing room between elements.
  </p>
</div>
