# Typography

Our typography system uses monospace fonts for a technical, precise aesthetic.

## Font Families

- **IBM Plex Mono**: Primary font for body text and UI elements
- **Departure Mono**: Display font for headings and emphasis

## Type Scale

The type scale provides consistent sizing across all text elements:

- **H1**: 2rem bold (Departure Mono)
- **H2**: 1.5rem bold (Departure Mono)
- **Body**: 1rem regular (IBM Plex Mono)
- **Small**: 0.875rem (IBM Plex Mono)

## Line Height

All text uses a comfortable 1.4 line height for optimal readability.
