## Examples

### Basic Card

A simple card with title, content, and footer.

<CodeExample html={`<m-card>
  <div slot="title">Card Title</div>
  <p>This is the main content of the card. It can contain any HTML elements.</p>
  <div slot="footer">Card footer</div>
</m-card>`} />

### Outline Variant

Use the outline variant for a bordered card style.

<CodeExample html={`<m-card data-variant="outline">
  <div slot="title">Outline Card</div>
  <p>This card has an outline variant with a border instead of a background.</p>
  <div slot="footer">Footer information</div>
</m-card>`} />

### Without Title or Footer

Cards can omit the title or footer slots.

<CodeExample html={`<m-card>
  <p>This card has no title or footer, just content.</p>
</m-card>`} />

### Subgrid Layout

Use subgrid to align card sections across multiple cards in a grid.

<CodeExample html={`<div class="collection">
  <m-card data-subgrid="true">
    <div slot="title">Card 1</div>
    <p>Short content.</p>
    <div slot="footer">Footer</div>
  </m-card>
  <m-card data-subgrid="true">
    <div slot="title">Card 2 with Longer Title</div>
    <p>This card has more content that spans multiple lines to demonstrate how subgrid keeps the footers aligned.</p>
    <div slot="footer">Footer</div>
  </m-card>
  <m-card data-subgrid="true">
    <div slot="title">Card 3</div>
    <p>Medium amount of content here.</p>
    <div slot="footer">Footer</div>
  </m-card>
</div>`} />

### Custom Styling

Cards support standard data attributes for customization.

<CodeExample html={`<m-card data-rounded="true" data-padded="true">
  <div slot="title">Styled Card</div>
  <p>This card has custom rounding and padding applied.</p>
</m-card>`} />

## Usage Notes

### Slots

- **`title`**: Optional slot for card title/heading
- **default**: Main content area
- **`footer`**: Optional slot for footer content

### Subgrid

When using `data-subgrid="true"`:
- The card must be a child of a grid container (e.g., `.collection` class)
- The card will span as many rows as needed based on which slots are used
- All cards in the grid will align their title, content, and footer sections

### Data Attributes

- **`data-rounded`**: Applies rounded corners
- **`data-padded`**: Applies padding to the card
- **`data-variant="outline"`**: Applies outline styling
- **`data-subgrid="true"`**: Enables CSS subgrid layout
