## Attributes

- **`tab`** (string): The currently active tab panel name
- **`label`** (string): Accessible label for the tab list (sets aria-label)
- **`position`** ("top"|"bottom"|"start"|"end"): Tab list orientation - top/bottom for horizontal, start/end for vertical

## Slots

- **tab**: Slot for `<m-tab>` elements (auto-assigned)
- **tab-panel**: Slot for `<m-tab-panel>` elements (auto-assigned)

## Events

- **`m-tab-show`**: Fired when a tab panel becomes visible. Detail: `{ tab: MTab, panel: MTabPanel }`
- **`m-tab-hide`**: Fired when a tab panel becomes hidden. Detail: `{ tab: MTab, panel: MTabPanel }`

## Keyboard Navigation

**Horizontal (top/bottom position):**
- **Arrow Left/Right** or **h/l**: Navigate between tabs
- **Home/End**: Jump to first/last tab

**Vertical (start/end position):**
- **Arrow Up/Down** or **j/k**: Navigate between tabs
- **Home/End**: Jump to first/last tab

## Accessibility

- Full ARIA support with proper roles and states
- Focus management follows ARIA tabs pattern
- Keyboard navigation matches native tab behavior
- Disabled tabs are properly announced to screen readers
