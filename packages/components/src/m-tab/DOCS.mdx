## Overview

Individual tab button used within `<m-tab-list>` for switching between tab panels.

## Attributes

- **`panel`** (string): ID of the associated tab panel to control
- **`active`** (boolean): Whether this tab is currently active
- **`disabled`** (boolean): Whether this tab is disabled and cannot be selected

## Slots

- **default**: Tab label content
