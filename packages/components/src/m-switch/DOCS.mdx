## Examples

### Basic Usage

A simple m-switch example.

<CodeExample html={`<m-switch example="test">
  Default content
</m-switch>`} />

## Attributes

- **`example`** (string): An example property

## Slots

- **default**: Default slot for component content

## Events

- **`m-switch-change`**: Fired when the example changes. Detail: `{ example: string }`
