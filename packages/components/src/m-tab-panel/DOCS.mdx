## Overview

Tab panel that displays content when its associated tab is active.

## Attributes

- **`name`** (string, required): Unique identifier for this panel, referenced by `<m-tab>`'s `panel` attribute
- **`visible`** (boolean): Whether this panel is currently visible (managed by `<m-tab-list>`)

## Slots

- **default**: Panel content

## See Also

- [`<m-tab-list>`](/components/m-tab-list) - Container that manages panels
- [`<m-tab>`](/components/m-tab) - Tab button that activates panels
